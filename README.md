# Simon Says Game

A fun and interactive Simon Says game built with HTML, CSS, and JavaScript. The game challenges players to memorize and repeat an increasingly longer sequence of colors. With each successful round, the sequence becomes more difficult, testing the player's memory and concentration.

# Features
.Interactive gameplay
.Random color sequence generation
.Increasing difficulty after each level
.Score and level tracking
.Responsive user interface
.Simple and engaging design

# Technologies Used
HTML
CSS
JavaScript (ES6)

# How to Play
.Press any key to start the game.
.Watch the color sequence carefully.
.Repeat the sequence by clicking the buttons.
.The sequence grows by one color after every successful round.
.The game ends when you click the wrong color.
